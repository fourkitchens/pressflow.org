We've done some house cleaning on Pressflow.org! Today we relaunched the site with more emphasis on our favorite part of Drupal: the community.

FAQs, Downloads, and Repos! Oh my!

We get a lot of recurring questions involving the release schedule, the status of Pressflow 7, and general compatibility, so we compiled the FAQ section. We're now linking to direct downloads of Pressflow 6, complete with a real-time indicator that reports the current version of Pressflow in the official Github repo. For contributors and the git-savvy, we also have more prominent links to the Pressflow 7 dev branch. For people needing support, there's a dedicated page to get you started.
